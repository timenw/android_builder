## Screenshot Specifications

### Phone Screenshots
- Resolution: 1080 x 1920 (minimum) or 1080 x 2340 (recommended)
- Format: PNG or JPEG
- Max size: 8MB each
- Minimum: 2 screenshots
- Recommended: 6-8 screenshots

### Tablet Screenshots (optional but recommended)
- Resolution: 1200 x 1920 or higher
- 7-inch and 10-inch tablet screenshots

## Design Notes for Screenshots

1. **Use a real device or emulator** — screenshots should show the actual app UI
2. **Use popular poems** — feature well-known poems like 静夜思 (Quiet Night Thoughts) or 春晓 (Spring Dawn)
3. **Show both languages** — make sure Chinese text and English translation are both visible
4. **Warm lighting** — the app uses a warm color palette (cream, indigo, terracotta) — make sure screenshots reflect this
5. **No personal data** — don't show real user notes or personal information
6. **Portrait orientation** — all screenshots should be in portrait mode

## Feature Graphic (required)
- Size: 1024 x 500 pixels
- Format: PNG or JPEG (no transparency)
- Should include: App name "300 Tang Poems", a tagline like "Poetry for the Soul", and visual elements from the app (moon, mountains, brush strokes)
- No screenshots in the feature graphic

## App Icon (required)
- Size: 512 x 512 pixels
- Format: 32-bit PNG with alpha channel
- Should be a simple, recognizable icon — consider a stylized moon with Chinese calligraphy elements
- Must be unique and not confusingly similar to other apps

## Promotional Video (optional)
- YouTube URL
- 30 seconds to 2 minutes
- Show the app's key features: daily poem, reading mode, search, quote cards
- Background music: traditional Chinese instrumental (guzheng or pipa)
